System Info visualizer (si-vis)

in development

Simple system info visualizer written in Rust for educational purposes only

Dependency resolution

  • eframe - official framework library for writing apps using egui, GUI library for Rust
  • egui_dock - library provides docking(tabs) support for egui
  • egui_extras - adds some features on top of egui, used for building table
  • sysinfo - crate used to get a system’s information

Code organization

  • si_gui, contains GUI app drawing and updating parts
    • si_data, stores and updates system info data
